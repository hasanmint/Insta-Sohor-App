Q:1. How does Javascript Work?

Answer: JavaScript is an interpreted language. Its also dynamically typed language. That means that it is a computer programming language that runs inside an Internet browser using V8 engine. As a interpreter in the browser reads over the JavaScript code, interprets each line by line with use just-in-time (JIT) compiler.into bytecode that the machine understands and can execute.

Q:2. What is Javascript event loop?

Answer: The event loop is the secret behind JavaScript's asynchronous programming. JS executes all operations on a single thread, but using few smart data structures, it gives us the illusion of multi-threading.The event loop is a constantly running process both the stack and the queue . If the call stack is not empty, the event loop waits until it is empty and places the next function from the callback queue to the call stack.

Q:3. What is the difference between localstorage and sessionstorage?

Answer:

Localstorage:
1.localstorage data is not cleared when the browser is closed.
2.localStorage objects to store key/value pairs .
3.localStorage data for a document loaded in a private browsing or incognito session

Sessionstorage:
1.sessionStorage data is cleared when the browser is closed.
2.sessionStorage also used objects to store key/value pairs.
3.sessionStorage data for a document is not loaded in a private browsing or incognito session
